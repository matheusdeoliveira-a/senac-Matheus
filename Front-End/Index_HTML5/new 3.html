Declaração

O nome das variáveis possuem regras para a sua declaração.
JavaScript é uma linguagem case sensitive.
Não pode conter caractere
Não pode iniciar com número
Não pode ter espaço
Pode iniciar apenas com o caractere_underline
Não pode ser com nomes de palavras reservadas da linguagem

Palavras Reservadas do JS

São as palavras chaves são identificadores pré reservados, como por exemplos o comandos da linguagem de programação
Exemplos de palavars reservadas: abstrat, byte, int, import, boolean, default, char, do, for, goto, case, continue, delete, etc...

Varíavel Composta ou Vetor

Varíavel é um tipo de estrutura de dados, onde a varíavel que consegue armazenar simultaneamente diferentes valores de um mesmo tipo.
Vetor é um tipo de estrutura de dados, onde a varíavel que consegue armazenar simultaneamente diferentes valores de um mesmo tipo.
Assim, vetor é uma varíavel composta, homogênea e unidimensional.
Em programação, as estruturas de dados são uma forma coerente e racional de organizar os dados e otimizar o seu uso. Um vetor unidimensional, ou um array, é uma das estruturas de dados mais comuns a serem aplicadas no desenvolvimento de um algoritmo poder ser considerado uma sequencia de posições de memória que é usada para armazenar um conjunto de dados do mesmo tipo e usando o mesmo nome.
Para acessar cada informação armazenada, separadamente, usa-se um índice que é representado por uma varíavel inteira. Essas informações são chamadas de elementos do vetor.
Exemplo:
notas = [5, 8, 10, 9]

Tipos de dados JS - Data Types

null - Não valor, ou inexistência de valor associado a uma varíavel
undefined - Conteúdo de varíavel não iniciada
boolean - Verdadeiro (true) ou Falso (false)
string - Para sequência de zeros e caracteres, normalmente representa textos
number - Números inteiros e flutuantes
NaN - Not a Number, número inválido
objetct - Para coleção de propriedades
symbol - Obsoleto usado para objetos
