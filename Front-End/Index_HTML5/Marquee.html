Texto animado dentro do marquee

Um recurso bem interessante para ser usado nos textos é o tag MARQUEE (letreiro) que permite que o texto tenha movimentos na horizontal e na vertical, sendo possivel ainda alterar sua velocidade, tamanho, cor de fundo, etc. esta é uma forma de dar detaque para alguns textos.



Scroll, faz com que o texto fique passado pela tela.



Slide, faz com que o texto entre por um lado e para do outro lado oposto.



Alternate, faz com que o texto fique alternando de um lado para o outro sem sair da tela.



BGCOLOR, define a cor de fundo do marquee.



DIRECTION = LEFT, faz o texto correr para a esquerda.



DIRECTION = RIGHT, faz o texto correr para a direita.



DIRECTION = UP, faz o texto correr para cima.



DIRECTION = DOWN, faz o texto correr para baixo.



HEIGHT=100 define a altura do marquee em valores de pixels.



HIDTH=100 define a largura do marquee em valores de pixels.



HIDTH=50% define a largura do marquee em valores percentuais.
HSPACE=100 define margens à esquerda e a direita do marquee.
VSPACE=100 define margens acima e abaixo do marquee.
LOOP=2 define quantas vezes o texto vai cruzar o marquee.
LOOP=INFINITE
SCROLLAMOUNT, define a velocidade do TEXTO e quanto maior o valor, mais rápido será o movimento.
